Getting Started

Clone carolayne repo

    
        git clone https://github.com/chrisenytc/carolayne.git

Enter in carolayne directory

    
        cd carolayne

Install dependencies

    
        npm install

Configure the settings in api/config

Build carolayne

    
        make build

Run carolayne

    
        npm start

Test your carolayne app

    
        npm test
